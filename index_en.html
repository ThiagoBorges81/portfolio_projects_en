Data Science Projects

Insights project for optimizing real estate purchases and sales

I used Python and its libraries for data analysis and statistics to detect properties with profit potential. Thus, properties could be bought by the company and sold with interesting profit margins. Purchase insights were based on property characteristics that include location, property size, number of rooms, among others. I built a web application using Streamlit and Streamlit Share to facilitate and optimize decision making by the company's stakeholders. The result of the presented solution, if implemented, would be an average profit of $71700.00 dollars per property.

Tools used in the project:

  • Python, Pandas, Numpy and Seaborn.
  • Jupyter Notebook.
  • Plotly and Folium interactive maps.
  • Streamlit Cloud.
  • Streamlit Python framework web.
  • Project details

Sales prediction and renovation budget

"End-to-end" Data Science project, where I applied several techniques including CRISP-DS, machine learning models for data processing and sales forecasting of a (fictional) pharmacy chain. After presenting a business problem by the company's CFO, the requested task was to forecast sales revenue for six weeks, based on a supervised machine learning procedure. After detailed data processing and sending the model to production, a bot was created in Telegram, where the CFO would be able to access the forecasts of each store, through his cell phone.

Tools used in the project:

  • Python, Pandas, Numpy, Seaborn, requests, inflection, xgboost, scipy, boruta, matplotlib.
  • Jupyter Notebook.
  • Sklearn.

Rowing Race Data ETL.

In this project, I collected information from more than 250 PDF documents, with data from World Rowing Championships. It was necessary to download the files from a public source, where the PDFs were stored, create codes in Python and use specific libraries to extract all the content of each PDF file. Once the data was extracted in string format, I carried out all the formatting and processing to use the data in academic research.
